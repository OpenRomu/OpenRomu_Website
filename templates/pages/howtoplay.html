Comment Jouer?

  • Créer un compte sur le serveur maître OpenRomu en vous rendant sur cette page
  • Télécharger la derniere version du jeu via le menu de gauche, en cliquant sur la plateforme de votre choix
  • Sur Windows & Linux, aucune installation n'est nécéssaire, dézippez l'archive ou vous le souhaitez.
    Sur MacOS, Copiez l'application OpenRomu.app dans votre dossier Applications.
  • Il vous suffit de lancer OpenRomu en utilisant l'executable du jeu.

Compte tenu de l'aspect récent du jeu, il est probable que les serveurs en ligne soient déserts.
Dans ce cas de figure, direction le discord (disponible sur la page d'accueil) pour trouver des partenaires de jeu!

Pensez a partager le jeu autour de vous pour que la fréquentation augmente.

Egalement, a l'heure actuelle, peu de maps sont fournies avec le jeu. Vous pouvez en créer vous même, ou porter des maps existantes vers OpenRomu !
Direction l'onglet mapping pour en savoir plus.

Bon jeu!
