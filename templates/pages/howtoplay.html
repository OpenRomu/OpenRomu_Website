Comment Jouer?

  • Créer un compte sur le serveur maître OpenRomu en vous rendant sur cette page
  • Télécharger la derniere version du jeu via le menu de gauche (Pour l'instant le jeu n'est disponible que pour Windows)
  • Aucune installation n'est nécéssaire, dézippez l'archive ou vous le souhaitez.
  • Aucun niveau n'a été crée spécialement pour OpenRomu, le jeu arrive donc sans aucune map jouable... Vous pouvez en créer vous même et les partager a la communauté!
    Ou vous pouvez télécharger les anciennes cartes de Romustrike en les téléchargeant depuis les archives ici
    Les cartes doivent se trouver dans un dossier appellé "maps", à coté du jeu. (Normalement, extraire l'archive là ou vous avez installé le jeu suffit.)
  • Il vous suffit de lancer OpenRomu en utilisant l'executable du jeu.


NOTE: Compte tenu que le jeu est récent, et n'est pas (encore?) plébiscité, il est probable que les serveurs en ligne soient déserts.
Dans ce cas de figure, direction le discord (disponible sur la page d'accueil) pour trouver des partenaires de jeu!
Pensez a partager le jeu autour de vous pour que la fréquentation augmente.

Bon jeu!
